# Webb Mock Fixture Standard

- Keep every HTML fixture under a single `<main>` element with semantic headings and sections.
- Include a `script type="application/json"` payload that captures the page's structured content.
- Do not use raw `<pre>` transcript dumps for HTML fixtures.
- Preserve source-faithful text even when reorganizing markup for readability.
- Keep course, faculty, athletics, and other extraction-heavy fixtures especially explicit and structured.
